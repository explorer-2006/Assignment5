# Assignment 5 – Responsive Webpage Design

This repository contains a **responsive webpage** created as part of **Assignment 5** for Web Development. The objective of the assignment was to design a modern webpage using **HTML5**, **CSS3**, and **Responsive Web Design (RWD)** techniques to ensure a seamless user experience across different screen sizes and devices.

---

## Objective

The goal of this assignment was to:

- Build a fully responsive webpage.
- Implement responsive layouts using CSS.
- Use media queries to adapt the design for different screen sizes.
- Create a clean and modern user interface with proper spacing and alignment.

---

## Features

- Responsive navigation bar
- Hero section with background image
- Latest articles section with cards
- About section
- Newsletter subscription form
- Responsive footer with social media icons
- Mobile-friendly navigation menu
- Adaptive layouts for desktop, tablet, and mobile devices

---

## Technologies Used

- HTML5

## Responsive Features

- Navigation links collapse into a menu icon on smaller screens.
- Article cards stack vertically on tablets and mobile devices.
- About section changes from a two-column layout to a single-column layout.
- Newsletter section adjusts spacing and input sizes for smaller screens.
- Footer layout adapts for better readability on mobile devices.

---

## How to Run

1. Clone or download this repository.
2. Open the project folder.
3. Launch the `index.html` file in any modern web browser.
4. Resize the browser window or use your browser's developer tools to view the responsive layout across different screen sizes.
